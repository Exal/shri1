Метод ценрирования с помощью абсолютного позиционирования и растягивания.
Достоинства:
Никакой дополнительной разметки, минимум стилей, работает для изображений.
Минусы: данный метод не работает в IE7 и ниже.Должна быть задана высота (см. Variable Height). Рекомендуется задать overflow: auto, чтобы контент не расползался. Не работает на Windows Phone
